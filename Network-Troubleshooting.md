# Network Troubleshooting

Step 1:
Check physical cable connection.

Step 2:
Verify IP configuration.

Linux:
ip a

Windows:
ipconfig

Step 3:
Ping localhost.

127.0.0.1

Step 4:
Ping gateway.

Step 5:
Ping public IP.

8.8.8.8

Step 6:
Check DNS resolution.

nslookup google.com

Step 7:
Check routing.

ip route

Step 8:
Check firewall settings.

Step 9:
Review logs.
