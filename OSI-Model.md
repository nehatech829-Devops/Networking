# OSI Model

OSI stands for Open Systems Interconnection.

The OSI model is a conceptual framework used to understand network communication.

It contains 7 layers.

## Layer 7 - Application

Provides services directly to users.

Examples:
- HTTP
- HTTPS
- FTP
- DNS
- SMTP

Functions:
- Web browsing
- Email
- File transfer

---

## Layer 6 - Presentation

Responsible for:

- Encryption
- Decryption
- Compression
- Data formatting

Examples:
- SSL/TLS

---

## Layer 5 - Session

Creates and maintains communication sessions.

Functions:
- Session establishment
- Session maintenance
- Session termination

---

## Layer 4 - Transport

Provides end-to-end communication.

Protocols:
- TCP
- UDP

Functions:
- Segmentation
- Reliability
- Error checking

---

## Layer 3 - Network

Responsible for routing.

Device:
- Router

Protocol:
- IP

Functions:
- Logical addressing
- Path selection

---

## Layer 2 - Data Link

Responsible for MAC addressing.

Device:
- Switch

Functions:
- Framing
- Error detection

---

## Layer 1 - Physical

Transmits raw bits.

Examples:
- Cables
- Fiber
- Wireless signals

Device:
- Hub
