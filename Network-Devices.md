# Network Devices

## Hub

A hub sends data to every connected device.

Characteristics:

- Layer 1 device
- No intelligence
- High collision domain

---

## Switch

A switch forwards data using MAC addresses.

Characteristics:

- Layer 2 device
- Intelligent forwarding
- Reduces collisions

---

## Router

A router connects different networks.

Characteristics:

- Layer 3 device
- Uses IP addresses
- Performs routing

---

## Access Point

Provides wireless connectivity.

Examples:

- Wi-Fi routers
- Enterprise APs

---

## Modem

Converts ISP signals into usable network signals.
