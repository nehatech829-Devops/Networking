# TCP/IP Model

TCP/IP is the practical networking model used on the Internet.

## Application Layer

Protocols:

- HTTP
- HTTPS
- DNS
- FTP
- SSH

Purpose:
Provides services to applications.

---

## Transport Layer

Protocols:

- TCP
- UDP

Purpose:
End-to-end communication.

---

## Internet Layer

Protocols:

- IP
- ICMP
- ARP

Purpose:
Routing packets between networks.

---

## Network Access Layer

Technologies:

- Ethernet
- Wi-Fi

Purpose:
Physical transmission of data.
